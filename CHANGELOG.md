## 2026-05-11 (2차)

### 대시보드 개선
- **판매처별 반품 추이 Legend 정렬**: 반품 수가 많은 판매처가 하단 라벨 가장 앞에 표시되도록 변경
- **판매 Top / 반품 Top 상품명 가시성 개선**: 상품명 컬럼을 굵은 차콜(#2d2d2d, bold) 텍스트로 변경
- **배송량 차트 이름 변경**: "월별 배송량" → "배송량"

### 물류 플로우 레이아웃 개선
- **Sankey 다이어그램 반응형**: SVG viewBox 적용으로 가로 스크롤 제거, 화면 너비 100% 활용
- **내부 스크롤 컨테이너 제거**: maxHeight 제한 및 overflow 제거, 자연스러운 뷰 제공
- **전체화면 버튼 제거**: 반응형 처리로 불필요해진 전체화면 버튼 제거

### CS 데이터 입력 기능 추가
- **CSV 일괄 업로드**: [상품] 컬럼 → 상품명, [반품 사유] 컬럼 → 반품 사유 자동 파싱
- 선택 컬럼: [날짜], [판매처] 자동 인식
- 오류 메시지에 헤더 목록 표시로 디버깅 편의성 향상

업데이트 내역을 날짜 역순으로 기록합니다.
